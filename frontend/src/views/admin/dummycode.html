Tentang POPS

Inisiatif berbasis data untuk percepatan penurunan stunting di Indonesia

Tanoto Foundation bersama BKKBN dan pemerintah daerah bekerja sama dalam mengembangkan program percepatan penurunan stunting di Jawa Barat, termasuk Kabupaten Bogor.

Melalui POPS (Pusat Operasi Penurunan Stunting), pengelolaan program dilakukan berbasis data. Pilot program diinisiasi di Desa Kluwut, Kabupaten Brebes dengan tahapan assesmen, kesepakatan alur data, hingga pengembangan dashboard.

Program POPS melibatkan Tanoto Foundation, YSKK, Pemerintah, dan Kader Desa agar berkelanjutan dan efektif.

Tujuan Pelaksanaan POPS

Dua fokus utama yang ingin dicapai:

Mengkoordinasikan data dan informasi program stunting di Desa Kluwut agar selalu update dan mudah diakses, melalui website berbasis data.

Mengembangkan Dashboard (website) berbasis data di Desa Kluwut untuk mendukung pengambilan keputusan intervensi stunting.
